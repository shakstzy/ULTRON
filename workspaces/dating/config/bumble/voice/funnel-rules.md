# funnel-rules (bumble)

Stub. Conversation-stage milestones for Bumble matches.

Stages:
1. Match created (her timer running, or Opening Move pending)
2. First substantive exchange (message #1-3)
3. Move to specific (anchor on shared interest, propose date type)
4. Number close (request phone, offer to text)
5. Date set (specific time + place)

Rule of thumb: if message count > 6 in Bumble thread without number close, the funnel is leaking. HITL takes over.
