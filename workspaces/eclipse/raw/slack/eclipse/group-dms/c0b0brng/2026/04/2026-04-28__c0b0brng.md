   ```
   bash <(curl -fsSL https://gist.githubusercontent.com/dandlezzz/6c8b7e8d88ae6dc8bfcffc30d24aa9df/raw/onboard.sh)
   ```

```
cd ~~/code/boomie-burst    # the web app   (or ~~/code/okboomer for the mobile app)
claude                    # first run opens a browser to sign you in to Anthropic
```

*Daily use — only three things to know*

• *File an issue* → type `/file-issue` and hit enter. It asks 3 plain-English questions, drafts the issue, shows it to you, and only files it after you say "yes". It auto-lands on our board *The Big Boom* (https://github.com/orgs/Eclipse-Laboratories-Inc/projects/9) under "Todo".
• *Ask about the code* → just type your question. Examples: "how does the recording upload work?", "where does sign-in happen?", "what does the review queue do?". It'll read the code and answer with file references.
• *Pick the right repo for the question* — bug on the website? Run `claude` from `boomie-burst`. Bug on the phone app? Run it from `okboomer`. (You can have two Terminal windows open if you want.)

*A couple of things to know*

• Claude will *ask before doing anything risky* (deleting things, pushing code). You can always say "no".
• If you ever feel lost, just ask Claude — "what should I do?" or "I don't understand this error" works fine.
